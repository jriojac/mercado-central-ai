# Diagramas del Proyecto

Esta carpeta contiene la documentación gráfica oficial del proyecto.

## Estructura

- Blueprint/
  Arquitectura general del sistema.

- Arquitectura/
  Diagramas de arquitectura por módulos.

- Flujos/
  Diagramas de procesos y flujos funcionales.

- Secuencia/
  Diagramas de secuencia e interacción entre componentes.

## Versionado

Los diagramas son documentos vivos y evolucionan durante el proyecto.

La versión vigente es siempre la de mayor número.

Las versiones anteriores podrán conservarse únicamente cuando representen cambios importantes de arquitectura.
