# Plantillas Oficiales del Proyecto

## Objetivo

Esta carpeta contiene las plantillas oficiales utilizadas para la creación de la documentación del proyecto **Mercado Central AI**.

El propósito es mantener una estructura homogénea, facilitar el desarrollo de nuevos módulos y garantizar la consistencia documental durante todo el ciclo de vida del proyecto.

---

## Plantillas disponibles

| Archivo | Uso |
|----------|-----|
| TEMPLATE_SDS.md | Diseño técnico del módulo |
| TEMPLATE_DOC.md | Documentación funcional |
| TEMPLATE_LOG.md | Registro de avances |
| TEMPLATE_MTR.md | Matriz de trazabilidad |
| TEMPLATE_CHANGELOG.md | Registro de Releases |
| TEMPLATE_README.md | README del proyecto |
| TEMPLATE_TEST_PYTEST.md | Pruebas automatizadas |

---

## Regla general

Toda nueva documentación deberá generarse utilizando estas plantillas, salvo que exista una justificación técnica aprobada durante una Auditoría Arquitectónica.
