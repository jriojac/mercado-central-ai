# DOC-011 -- Estándares de Desarrollo

## Objetivo

Definir reglas obligatorias para mantener un proyecto consistente,
seguro y mantenible.

## Estándares

-   Python 3.12.x
-   snake_case para funciones
-   PascalCase para clases
-   Docstrings obligatorios
-   Manejo de excepciones
-   .env y .gitignore obligatorios

## Seguridad

-   Nunca subir secretos.
-   Validar entradas.
-   No registrar datos sensibles.
-   Revisar dependencias antes de la entrega.

## Documentación

Todo módulo debe tener documento, diagrama, pruebas y trazabilidad.
