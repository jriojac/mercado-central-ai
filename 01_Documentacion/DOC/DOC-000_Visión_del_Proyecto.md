# DOC-000 -- Visión del Proyecto

**Proyecto:** Challenge Alura -- Agente IA Mercado Central 24h\
**Versión:** 1.0\
**Fecha:** 30/06/2026

## 1. Propósito

Desarrollar un Agente Inteligente basado en IA Generativa capaz de
consultar, interpretar y responder preguntas utilizando la documentación
oficial de Mercado Central 24h mediante técnicas RAG.

## 2. Problema

La información corporativa está distribuida en múltiples documentos
(FAQ, reglamentos, políticas, manuales, inventario y directorios),
dificultando el acceso rápido y consistente.

## 3. Solución

Construir un agente conversacional apoyado en una base de conocimiento
vectorizada que responda utilizando únicamente información oficial.

## 4. Usuarios Objetivo

### Clientes

-   Consultas generales
-   Devoluciones
-   Sucursales
-   Horarios
-   FAQ

### Proveedores

-   Alta
-   Compras
-   Entregas
-   Políticas

### Colaboradores

-   Reglamento
-   Procedimientos
-   Operación

## 5. Beneficios

-   Centralización del conocimiento.
-   Atención 24/7.
-   Respuestas consistentes.
-   Escalabilidad.

## 6. Alcance Inicial (MVP)

-   Consultar documentación.
-   Consultar inventario.
-   Consultar sucursales.
-   FAQ.
-   Cambios y devoluciones.
-   Información para proveedores.

## 7. Tecnologías

-   Python
-   LangChain
-   Gemini 2.5 Flash
-   ChromaDB
-   Streamlit
-   GitHub
